

 # ELMenus Order Management System

Smple Console application GUI order menu system, for one of eygpt largest companies.

 ## Building

Use cmake to generate the build file, then use your own building system, i.e **Makefile** to build the system. 

<code>cmake -B build -S .</code> <br>
<code>make -C build</code>
